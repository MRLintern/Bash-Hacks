## compile

This bash script compiles some C++ files into an executable. Essentially just a `Makefile`.

### How to run

- `$ chmod +x compile.sh`
- `$ ./compile.sh`

### Sample Output

- 

- `$ ./compile.sh`
- `Compilation successful. Executable created: bin/main`
- `$ ls`
- `add.h  bin  compile.sh  main.cpp`
- `cd bin`
- `main`
- `$ ./main`
- `The sum of 12 & 5 is 17`
